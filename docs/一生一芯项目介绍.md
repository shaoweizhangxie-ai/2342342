# 一生一芯项目介绍

## 项目概览

“一生一芯”（One Student One Chip，YSYX）是一项面向处理器芯片人才培养的开源、公益性教学实践计划。项目由中国科学院大学计算机科学与技术学院联合中国科学院计算技术研究所科研工程支撑团队于 2019 年 8 月启动，尝试把课堂知识、系统实验与真实芯片设计和流片贯通起来。

项目最具代表性的愿景是：**让学生带着自己设计的处理器芯片毕业。** 它并不只要求学习者者写出一段可以仿真的硬件代码，而是希望学习者理解程序怎样在处理器上运行，并逐步接触处理器前端设计、SoC 集成、后端物理设计、流片、回片测试和系统软件等环节。

## 为什么需要“一生一芯”

传统计算机教育往往把程序设计、组成原理、体系结构、操作系统、编译原理和集成电路设计分成不同课程。学生能够学到许多知识点，却不一定有机会把它们放进一个真实系统中反复验证。

“一生一芯”针对的正是这种“会知识但缺少完整实践”的问题。项目以开源处理器为载体，用芯片敏捷开发方法降低实践门槛，通过完整项目把计算机科学（CS）与电子信息工程（EE）的知识连接起来，培养兼具软件和硬件视角的计算机系统人才。

## 核心理念

### 1. 理论与实践贯通

学习者不仅要知道指令、流水线、缓存或异常机制的定义，还要亲手实现、验证和调试处理器，在工程问题中理解原理。

### 2. 软件与硬件协同

项目强调从程序到电路的完整链路：程序经过编译和运行时环境，最终由处理器执行；处理器设计又要接受应用、操作系统和测试程序的检验。

### 3. 从设计走向硅实现

“流片”是项目的重要特色。符合条件的设计可以进入 SoC 集成和物理设计流程，并提交晶圆厂制造。回片后还要在板卡上完成测试、调试和演示，使设计不止停留在仿真阶段。

### 4. 开源与公益

项目依托开源软硬件生态开展教学，报名和学习免费，面向不同学校、年级和专业的学习，也允许零基础学习者从预学习开始。根据官网说明，因经费有限，免费流片机会仅面向在校生，并且需要满足项目当期的考核和选拔要求。

### 5. 独立解决问题

教学过程强调主动探索。助教通常在关键节点提供方向性指导，而不是直接给出答案；阅读文档、定位故障、设计实验和复盘错误本身就是培养目标的一部分。

## 学习内容与路径

项目会随期次调整任务和考核，整体上可理解为以下递进路线：

1. **预学习与环境准备**：补充 Linux、命令行、编程、数字电路等基础，建立基本工程能力。
2. **计算机系统基础实践**：通过软件与硬件实验理解程序执行、指令集、模拟器、运行时环境和调试方法。
3. **处理器前端设计**：围绕 RISC-V 指令集实现处理器，持续完成功能验证、性能改进与工程化调试。
4. **系统能力建设**：让处理器运行更完整的软件栈，从软硬件协同角度理解异常、中断、存储系统、外设和操作系统。
5. **SoC 与后端实践**：部分达到要求的学习者继续参与 SoC 集成、物理设计、板卡测试和系统调试，推动设计走向流片。

具体任务、处理器实现要求、认证等级和流片标准，应以当期官方文档为准。

## 学习者能获得什么

- 对计算机系统建立从应用、系统软件、指令集到数字电路的整体认识；
- 获得处理器设计、仿真、验证、调试和性能分析经验；
- 锻炼阅读文档、使用工程工具和独立解决未知问题的能力；
- 理解前端设计、SoC、EDA 后端流程、流片和板级测试之间的关系；
- 接触开源社区的协作方式，并有机会参与相关生态建设；
- 达到相应要求后，可申请项目认证；在校生还可能获得教学流片机会。

## 适合哪些人

“一生一芯”适合对以下方向有长期兴趣的学习者：

- 处理器设计与计算机体系结构；
- 操作系统、编译器与底层系统软件；
- 数字电路、SoC 与集成电路设计；
- RISC-V 与开源芯片生态；
- 希望通过大型实践项目建立完整计算机系统观的人。

项目允许零基础加入，并不意味着学习过程轻松。参与者需要投入较长时间，适应开放式问题和频繁调试，也需要对独立学习保持耐心。

## 项目的价值

“一生一芯”的价值不只在于最终得到一颗芯片，更在于建立一种贯通式培养路径：以真实处理器为主线，把原本分散的课程知识连接成可运行、可验证、可流片的系统。对学习者而言，它把“理解计算机如何工作”变成了一段可以亲手构建和检验的工程经历；对教育而言，它探索了依托开源生态培养软硬件全栈人才的新方式。
